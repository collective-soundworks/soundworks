(async) start()

Source:
  • client/Plugin.js, line 59

This method is called during the pluginManager.start() which is itself called during server.init(). At this point the state manager is ready to use.

Example
class MyDummyPlugin extends Plugin {
  async start() {
    this.state = await this.server.stateManager.create(`s:${this.id}`);
    await new Promise(resolve => setTimeout(resolve, 1000));
  }
}
