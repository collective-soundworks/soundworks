StateManager

client. StateManager

Component dedicated at managing distributed states, accessible through common.SharedState instances, among the application.

An instance of StateManager is automatically created by the soundworks.Client at initialization (cf. client.Client#stateManager).

Tutorial: https://collective-soundworks.github.io/tutorials/state-manager.html

(async) observe(callback)

Source:
  • client/StateManager.js, line 205

Observe all the common.SharedState instances that are created on the network.

Example
this.client.stateManager.observe(async (schemaName, stateId, nodeId) => {
  if (schemaName === 'something') {
    const state = await this.client.stateManager.attach(schemaName, stateId);
    console.log(state.getValues());
  }
});
Parameters:
Name Type Description
callback client.StateManager~observeCallback

Function to be called when a new state is created on the network.
Returns:

Promise - Return a Promise that resolves when the callback as been executed for the first time. The promise value is a function which allows to stop observing the network.
