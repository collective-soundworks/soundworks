(async) start()

Source:
  • common/BasePlugin.js, line 70

Start the plugin. This method is automatically called during the pluginManager.start() which is itself called during the init lifecyle step. After this point the plugin should be ready to use.

Example
class MyDummyPlugin extends Plugin {
  async start() {
    this.sharedState = await this.server.stateManager.create(`s:${this.id}`);
    await new Promise(resolve => setTimeout(resolve, 1000));
  }

  async stop() {
    await this.sharedState.delete();
  }
}
