Class: Placer

soundworks/client~Placer

Interface for the 'placer' service.

This service is one of the provided services aimed at identifying clients inside the experience along with the 'locator' and 'checkin' services.

The 'placer' service allows a client to choose its location among a set of positions defined in the server's setup configuration entry.

The service must be used with its server-side counterpart

Constructor

new Placer(options)

WARNING This class should never be instanciated manually

Parameters:
Name Type Description
options Object
Properties
Name Type Attributes Default Description
mode String <optional>
 'list'

Sets the interaction mode for the client to choose its position, the 'list' mode proposes a drop-down menu while the 'graphic' mode (which requires located positions) proposes an interface representing the area and dots for each available location.
