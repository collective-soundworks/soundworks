SharedStateManagerClient

client. SharedStateManagerClient

Component dedicated at managing distributed states, accessible through common.SharedState instances, among the application.

An instance of SharedStateManagerClient is automatically created by the soundworks.Client (cf. client.Client#stateManager).

Tutorial: https://collective-soundworks.github.io/tutorials/state-manager.html

Source:

Observe all the common.SharedState instances that are created on the network.

Example
this.client.stateManager.observe(async (schemaName, stateId, nodeId) => {
  if (schemaName === 'something') {
    const state = await this.client.stateManager.attach(schemaName, stateId);
    console.log(state.getValues());
  }
});
