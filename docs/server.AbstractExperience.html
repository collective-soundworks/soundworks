AbstractExperience

server. AbstractExperience

Base class to extend in order to create the server-side counterpert of a soundworks client.

The user defined Experiences are the main components of a soundworks application.

Source:

Require a registered plugin, all client types associated to the experience will also be associated to the required plugin. Requiring a plugin should always be done between soundworks.init and soundworks.start. In most case this method is called in the constructor the Experience.
